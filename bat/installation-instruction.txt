1. Download Python & Add to environment variables PATH (https://www.python.org/downloads/release/python-3115/)
2. create .env file in the same directory as main.py based on .env.example
3. Edit install.bat with notepad, adjust line 2 to "cd <path>\<to>\<AI>\<directory>". Example: cd C:\Users\gundu\Documents\gundu-ai\gundu-ai
4. run install.bat
5. Edit gundu.bat with notepad, adjust line 2 to "cd <path>\<to>\<AI>\<directory>". Example: cd C:\Users\gundu\Documents\gundu-ai\gundu-ai
6. run gundu.bat
